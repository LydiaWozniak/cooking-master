## Installation and Setup Instructions

You will need `node` and `npm` installed globally on your machine and then you can clone this repository.  

Installation:

`npm install`  

To Run Test Suite:  

`npm test`  

To Start Server:

`npm start`  

To Visit App:

`localhost:3000`
